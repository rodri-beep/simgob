# Personajes del tutorial

Coloca aquí los retratos de los dos asesores que guían el tutorial
("¿Cómo funciona?"). Mientras no existan, el modal muestra un avatar de
respaldo con la inicial.

Nombres de archivo esperados (formato **WebP**):

- `asesora.webp` — la asesora (Elena)
- `asesor.webp` — el asesor de Hacienda (Diego)

Recomendado: imagen cuadrada (p. ej. 256×256 o 512×512), estilo pixel-art
coherente con la estética del juego. Activos originales (sin marcas de terceros).
